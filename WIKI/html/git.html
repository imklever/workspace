git repo 迁移

step1. 获取需要迁移的repo 

root@localhost:~# git clone --bare git@github.com/old_repo.git

step2. 推送至新的目标repo: 

root@localhost:~# cd old_repo.git
root@localhost:~# git push --mirror git@github.com/new_repo.git
